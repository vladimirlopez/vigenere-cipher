# Vigenère Cipher Simulation

An interactive Vigenère Cipher simulation designed for proper educational scaffolding of polyalphabetic substitution.

## Educational Features
- **Interactive Tabula Recta**: Visualizes the intersection of Key rows and Plaintext columns.
- **Walkthrough Mode**: Forces step-by-step engagement to prevent passive use.
- **Keyword Expansion**: Visually demonstrates the cyclic nature of the keyword.
- **Decode Mode**: Teaches the reverse process of finding the ciphertext in the row and tracing back to the column header.

## Tech Stack
- HTML5 / CSS3 (Vanilla)
- JavaScript (ES6+)
- Vite (Build tool)
